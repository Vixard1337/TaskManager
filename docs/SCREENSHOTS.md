# Screenshot Checklist

Use this checklist when preparing section 5 of the report.

## Required screenshots

- [ ] Home page (`/Index`)
- [ ] Users list (`/Users/Index`)
- [ ] Create user form (`/Users/Create`)
- [ ] Tasks list (`/Tasks/Index`)
- [ ] Tasks list with active tag filter (`/Tasks/Index?Tag=...`)
- [ ] Create task form (`/Tasks/Create`)
- [ ] Edit task form (`/Tasks/Edit?id=...`)
- [ ] Delete confirmation (`/Tasks/Delete?id=...`)
- [ ] Tasks list showing Done/Undo state change

## Screenshot naming convention

- `01-home.png`
- `02-users-index.png`
- `03-users-create.png`
- `04-tasks-index.png`
- `05-tasks-filter-tag.png`
- `06-tasks-create.png`
- `07-tasks-edit.png`
- `08-tasks-delete.png`
- `09-tasks-done-undo.png`

## Notes

- Use one browser language consistently.
- Keep the same window size for all screenshots.
- Ensure sample data is visible (at least 2 users and 3 tasks).
